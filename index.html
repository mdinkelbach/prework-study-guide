  • The "body" element represents the visible content shown to the user.
  • The "header" element within the "body" element is different than the "head" element.

Git

  • git clone [url]: obtain a repository from an existing URL
  • git status: tells you which branch you are currently on
  • git pull [origin {branch}]: takes files from selected branch on GitHub and puts them on your local PC. origin refers to Github, {branch} (usually main for main branch) refers to the branch you want to copy.
  • git checkout [branch name]: switches to the selected branch. Can be done with -b {branch name} to create and name a new branch, then switch to it.
  • git add [file name or -A for all]: stages the selcted changes to be added to a commit.
  • git commit: creates a version history snapshot.
  • git push: sends the commited (or snapshotted) changes to GitHub for a pull request.
